# Online-item-price-tracking-system.
โครงงานวิจัยนี้เป็นการศึกษาหลักการทำงานหรือขึ้นตอนในระบบว่าการดึงข้อมูลมาจากแหล่งข้อมูลอื่นนั้นสามารถทำให้ติดตามราคาขึ้นลงตลอดเวลาเพื่อที่จะแสดงช่วงราคาขึ้นลงของราคาปกติ การพัฒนาระบบการติดตามราคาสินค้าขึ้นมาเพื่อให้ผู้ใช้สามารถตัดสินใจช่วงเวลาที่เหมาะสมกับการซื้อสินค้าได้
